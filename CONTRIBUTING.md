# Contributing to Flakiness Experiment Framework

Thank you for considering contributing!

## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Add tests if applicable
5. Check your result

## Reporting Issues

Please include:
- Steps to reproduce
- Expected behavior
- Actual behavior
- Logs or screenshots (if applicable)

## Coding Standards

- Follow PEP8
- Use meaningful variable names
- Add docstrings to functions
